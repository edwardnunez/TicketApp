Para arrancar la aplicación, necesitas lo siguiente:

- Docker
- VSCode (u otro editor similar)
- MongoDB
- Node.js

1. Se debe ejecutar npm install en la carpeta base del proyecto
2. se debe ejecutar el comando *docker-compose up --build* en la carpeta base del proyecto
3. Si diera algún error, se debe ejecutar npm install en la carpeta de frontend y en la de cada servicio respectivamente
4. Después de eso, volver a ejecutar *docker-compose up --build* en la carpeta base del proyecto
5. La aplicación estará disponible en http://localhost:3000
